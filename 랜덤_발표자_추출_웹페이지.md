# 랜덤 발표자 추출 웹페이지

## 개요

수업시간에 발표자를 랜덤을 뽑아서 보여주는 웹페이지

---

## 목표 및 대상

초등학교 지체장애 특수학생 

---

## 핵심 기능

- 발표자를 랜덤으로 뽑아서 보여주는 기능
  - 추출 인원을 선택할 수 있는 기능
- 학생명단을 추가하는 기능
  - 한 명 혹은 여러명을 일괄로 추가할 수 있는 기능
- 교사가 마음대로 몰래 발표자 순서를 정하는 기능

---

## 화면 구성

발표자를 보여줄 때, 슬롯머신처럼 재밌게 보여주는 화면

---

## 기술 스택

- 프론트엔드
  - react + vibe

---
